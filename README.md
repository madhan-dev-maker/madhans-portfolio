# Portfolio (vanilla)

Simple portfolio inspired by modern developer portfolio layouts (dark theme, neon accents, motion, and scanline-style UI).

## Files

- `index.html` - page markup
- `styles.css` - styling + animations
- `script.js` - background, typing effect, project grid, skills bars, and mailto contact form

## Run locally

If you have Python:

```powershell
cd "F:\Dev Dynsty\AI"
python -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize

- Change your name/title/email/social links in `index.html`
- Update the project cards array in `script.js`
- Adjust colors in `styles.css` under `:root`
